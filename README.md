Project: Cardiovascular Disease Prediction
Cardiovascular disease is one of the leading causes of death worldwide. This project applies data preprocessing, exploratory analysis, and visualization techniques to a heart disease dataset, followed by building predictive models.

Key steps include:

🔍 Data Preprocessing: Cleaning and preparing the dataset for analysis.

📊 Visualization & Insights: Generating plots and correlation matrices to understand feature relationships.

🤖 Model Training: Implementing multiple machine learning algorithms — SVM, KNN, Decision Trees, Logistic Regression, and Random Forest.

📈 Performance Comparison: Evaluating accuracy levels across models to identify the best approach.

🩺 Final Model: Deploying the most accurate model for reliable heart disease detection.

This project demonstrates how machine learning can support early diagnosis and prevention of cardiovascular disease.
